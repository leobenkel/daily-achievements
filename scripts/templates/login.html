Github Credentials

This application use a github repo as a database.

This is a client side only app, hosted by Netlify. There are no servers at all.

Feel free to look at the network tab in the chrome dev console. Nothing is travelling anywhere. You own your data. You can even clone the database created and modify the database yourself if you'd like (Link at the bottom of the page when you fill up the form, it needs the name and username).

And the whole app is open source if you want to double check: https://github.com/leobenkel/daily-achievements

To connect, choose a 'name', that will be used to create a database for you, which allow you to have several account under the same Github credentials.

Then, go to the github setting page: https://github.com/settings/tokens and create a personal token. You can destroy it when you don't want to use it anymore, since the database is a repo under your name, you won't lose anything. And make sure to only select 'repository' permission so you are safe.

If you are scared, I personally created a brand new Github account with nothing in it but the database.

You can then paste you Github username and Github token in the form and press 'connect'.
